# MOX Start

A basic configuration of the Turris MOX router. Marvell Armada 3720 1 GHz dual
core, 1024 MB DDR3 RAM, GWAN/GLAN, 1× USB 3.0 (5 V, 1.5 A on output), 1× microSD
slot
